    Nonebot-plugin-novelai

    A description of AI drawing plug-in for Nonebot

    DocsUpdate

    Support multiple back ends

    Novelai·Naifu·SD-Webui

    Load leveling

    If you have multiple servers, you can use them at the same time.

    Advanced command

    Specify negative entries, specify any parameters, even the model used by SD

    Custom reply format

    You can replace the reply with Japanese, English, cute words, or even define it yourself.

    Support extension

    Other plug-ins can generate images through simple calls without knowing the internal implementation of AI painting.

    Simulate point calculation mode

    Simulate the official point calculation model to manage users and control user behavior more accurately.

    Chinese support

    Built-in translation without API, automatically convert TAG into English.There are also optional engines that require API.

    Rich additional features

    Query the TAG of pictures, check other people's XP, random girls, etc.
